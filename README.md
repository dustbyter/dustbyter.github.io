# STEM-X (dustbyter.github.io)
STEM-X is a free, gamified website hosted on Github to decrease the gap in STEM education. With a bit of research, we were able to come to the conclusion that the biggest gaps in STEM come from math and in that weird transition period between elemntary school and middle school, so through this website, we built a database with an emphasis on math from the 5th and 6th grades. However, we did not skimp out on the other aspects of STEM, also including elements of science, math and technology. 

While this is a beginner project, we would like to expand on it to include more content, a bigger database, and hopefully, a more engaging platform. If we had more time on this project, we would probably include a better storyline, with more characters and hopefully, something to do with the potions. And for the STEM aspect, ranging difficulty based on grade level from K-12, with a bigger database of at least 100-200 questions for each grade, with sections so people can choose what STEM aspect they would like to focus on.
